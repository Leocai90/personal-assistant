# 个人助理 - Personal Assistant

一个现代化的个人助理应用，帮助你记录灵感、管理待办事项，并提供智能提醒功能。

## 功能特性

### 灵感板
- 快速记录瞬间的灵感和想法
- 分类管理（通用、项目、学习、生活、工作）
- 支持标签系统
- 编辑和删除功能
- 按时间排序展示

### 待办事项
- 创建和管理日常任务
- 设置优先级（高、中、低）
- 添加截止日期和提醒日期
- 任务完成状态切换
- 多维度筛选和排序
- 可视化统计信息

### 智能提醒
- 自动检测即将到期的任务
- 逾期任务提醒
- 高优先级任务提示
- 自定义提醒时间
- 分类查看不同类型的提醒

### 仪表板
- 统计概览（待处理、已完成、完成率）
- 快速访问主要功能
- 直观的数据展示

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI样式**: Tailwind CSS
- **图标库**: Lucide React
- **日期处理**: date-fns
- **数据存储**: LocalStorage

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 `http://localhost:3000` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用指南

### 记录灵感

1. 点击"新想法"按钮
2. 填写标题和详细内容
3. 选择合适的分类
4. 添加相关标签（可选）
5. 点击保存

### 管理待办事项

1. 点击"添加新任务"
2. 输入任务标题和描述
3. 设置优先级
4. 选择截止日期和提醒日期
5. 点击保存
6. 点击圆形按钮标记任务完成

### 查看提醒

1. 在仪表板顶部会显示智能提醒面板
2. 或点击导航栏的"智能提醒"查看所有提醒
3. 提醒包括：到期提醒、即将到期、已逾期、高优先级

## 数据存储

所有数据都保存在浏览器的 LocalStorage 中，包括：
- 灵感记录 (assistant_ideas)
- 待办事项 (assistant_todos)
- 提醒设置 (assistant_reminders)
